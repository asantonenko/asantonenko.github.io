Константні вказівники та вказівники на константні дані

Використання модифікатора const з вказівниками має особливість. Нагадаємо, що цей модифікатор вказує, що значення змінної не може бути змінене після ініціалізації.

Для вказівників існує два типи константності:

  1. Сам вказівник може бути константним (не можна змінити адресу, на яку він вказує).
  2. Вказівник може вказувати на константні дані (забороняє змінювати значення через цей вказівник).

Приклад з двома модифікаторами const:

int x = 5, y = 7;
const int * const ptr = &x;

Як це працює:

  • const ліворуч від зірочки: забороняє зміну даних ((*ptr)++ або *ptr = 2);
  • const праворуч від зірочки: забороняє зміну адреси вказівника (ptr = &y).

Типи вказівників:

  • int* — звичайний вказівник (можна змінювати дані та адресу);
  • const int* — вказівник на константу (дані не змінюються, адресу — можна);
  • int* const — константний вказівник (адресу не змінити, дані — можна);
  • const int* const — константний вказівник на константу (нічого не можна).

Важливо: Запис int const* еквівалентний const int*. Позиція const ліворуч від зірочки визначає його дію.
