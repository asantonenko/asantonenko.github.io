Но все же, продвинутый калькулятор умеет выполнять не только 4 арифметические операции, но и рассчитывать значения различных функций. Давайте попробуем рассчитать значения функций для некоторого аргумента.

Чтобы запустить некоторую функцию в языках C и C++, нужно указать имя функции и в скобках её аргументы или параметры. Например, рассмотрим сопоставление некоторым математическим выражениям их эквивалентов в C/C++:

Математическое выражение Выражение на языках C/C++
\(2x\) 2*x
\({x+y \over xy}\) (x+y)/(x*y)
\(\sqrt{x}\) sqrt(x)
\(x ^ y\) pow(x,y)

Функция pow имеет два параметра, которые отделяются друг от друга запятой.

Итак, напишем калькулятор, который будет вычислять функции одного аргумента. Для начала введем с клавиатуры аргумент функции, который назовём x:

#include <iostream>  //заголовочный файл для ввода и вывода

using namespace std; //используем пространство std

int main()    //основная функция программы
{             //начало основного блока                 
    double x; //Аргумент (параметр) функций  

    cout << "Введите аргумент функций (вещественное число):" << endl; //Выводим подсказку
    cin >> x;                                                         //Вводим значение с клавиатуры

    //Здесь будет остальная часть программы

    return 0;     //сообщаем, что программа успешно завершила работу
}                 //конец основного блока

Теперь добавим вывод рассчитанных значений разных функций:  

#include <iostream>
#include <cmath>     //заголовочный файл для математических функций

using namespace std; 

int main()    
{                           
    double x; 
    cout << "Введите аргумент функций (вещественное число):" << endl; 
    cin >> x;                                                         
    
    cout << "Квадрат           " << x << " = " << x*x      << endl;  //Печатаем квадрат x
    cout << "Куб               " << x << " = " << pow(x,3) << endl;  //Печатаем куб x
    cout << "Квадратный корень " << x << " = " << sqrt(x)  << endl;  //Печатаем квадратный корень x
    cout << "Экспонента        " << x << " = " << exp(x)   << endl;  //Печатаем экспоненту x
    cout << "Натурал. логарифм " << x << " = " << log(x)   << endl;  //Печатаем натуральный логарифм x
    cout << "Десятич. логарифм " << x << " = " << log10(x) << endl;  //Печатаем десятичный логарифм x    
    cout << "Синус             " << x << " = " << sin(x)   << endl;  //Печатаем синус x
    cout << "Косинус           " << x << " = " << cos(x)   << endl;  //Печатаем косинус x
    cout << "Тангенс           " << x << " = " << tan(x)   << endl;  //Печатаем тангенс x
    cout << "Арксинус          " << x << " = " << asin(x)  << endl;  //Печатаем арксинус x
    cout << "Арккосинус        " << x << " = " << acos(x)  << endl;  //Печатаем арккосинус x
    cout << "Арктангенс        " << x << " = " << atan(x)  << endl;  //Печатаем арктангенс x

    return 0;     
}

Напомню, что в коде выше прокомментированы только новые строки. Для использования математических функций нужно подключить заголовочный файл cmath (или заголовочный файл math.h в языке C). Как Вы видите стандартных функций квадрат и куб в языках C/C++ нет, но можно записать квадрат и куб через функцию pow как вторую и третью степень: pow(x,2) и pow(x,3) или просто через умножение как x*x и x*x*x. Также отмечу, что в языках C/C++ для некоторых функций используются обозначения отличные от общепринятых в математике (создатели языка Си плохо помнили стандартные математические обозначения?). Кстати, я не стал заводить по переменной для хранения результатов вычисления функций – иначе было бы очень много переменных в программе. Вместо этого я просто сразу вывел вычисленные значения в поток вывода cout.
