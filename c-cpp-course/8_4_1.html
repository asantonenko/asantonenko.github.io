Методи класу string

Класи в об’єктно-орієнтованих мовах об’єднують не лише дані (як структури мови Сі, наприклад), а й підтримують операції (дії) над ними. Частина операцій реалізована у вигляді перевантажених операцій мови Сі. Це означає, що операція мови Сі при застосуванні до екземплярів класу позначає щось інше, не зовсім те, що й у мові Сі над звичайними даними. Такими прикладами перевантажених операцій для класу string є + і =, а також операція +=, яка поєднує можливості цих двох операцій. Також операція [ ] вибирає певний символ рядка (нумерація починається з нуля). Звичайно, не всі дії можна виразити через оператори для класів. Те, що в мові Сі, а також в інших структурно-процедурних мовах реалізується за допомогою функцій (процедур, підпрограм в інших мовах), в об’єктно-орієнтованих мовах часто реалізується як методи (method або member function). Метод схожий на звичайну функцію, лише застосовується до певного об'єкта та взаємодіє з даними (полями) об'єкта, а також може викликати інші методи для того ж чи іншого об'єкта.

Виклик методу виглядає так:

ім’я_об'єкта.ім’я_методу(параметри_методу)

Методи можна поділити на запити і команди. Методи-запити не змінюють об'єкт, а повертають певну інформацію про нього. Наприклад, метод length повертає довжину рядка. Він не має додаткових параметрів (сам рядок не є явним параметром, а до нього застосовується цей метод). Приклад:

string str1 = "help";

int len = str1.length();

У цьому прикладі цілій змінній len буде присвоєно результат застосування методу length до рядка str1, який дорівнюватиме 4. Зверніть увагу, що сам рядок str1 не є параметром методу, а є об’єктом, до якого метод застосовується.

Методи-команди, навпаки, змінюють об’єкти. Методи-команди зазвичай не повертають результат, тобто мають тип результату void. Прикладом методу, що змінює рядок, є метод erase, один з варіантів якого має два параметри: перший — початкова позиція видаляємої підрядка, другий — довжина. Наприклад, str.erase (1,1) видалить другий символ (символ з індексом 1). Наступний приклад видалить символи з позицій від 5 до 10:

string s="0123456789ABC";

s.erase(5, 6); //від 5-ї до 10-ї позиції 6 символів

cout <<s <<endl; //01234BC

Якщо ви переглянете на сайті www.cplusplus.com опис методу erase, то побачите, що його запис трохи складніший, а саме його заголовок має вигляд (див. http://www.cplusplus.com/reference/string/string/erase/):

string& erase (size_t pos = 0, size_t len = npos);

Навчимося читати такий запис. Спершу вказано тип повернення — як бачимо, це не void. Метод повертає деякий об’єкт класу string за посиланням. У секції Return value описано, що ця функція (точніше, перша її версія sequence (1)) повертає сам рядок, до якого застосовується метод. Це досить звична практика для C++: дуже часто, якщо метод-команда за своєю логікою не має що повертати, він повертає сам об’єкт за посиланням. Зазвичай цей результат ігнорується та не використовується, як у нашому попередньому прикладі. Далі вказується ім’я методу erase. Далі в дужках вказано два параметри — pos і len — позиція першого видаляємого символу (нумерація з нуля) і кількість символів на видалення. Це можна прочитати у секції Parameters. Тип цих параметрів — size_t, це беззнаковий цілий тип. Записи = 0 і = npos — це значення параметрів за замовчуванням. Тобто можна не вказувати обидва аргументи, а лише перший (або взагалі не вказувати), наприклад:

string str1="abcd", str2="ef";

str1.erase(2); //видалити починаючи з символу з індексом 2

//(тобто третього) до кінця рядка

str2.erase(); //видалити увесь рядок (рядок стане порожнім)

cout <<str1<<","<<str2<<"."<<endl; //буде надруковано: ab,.

У цьому методі константа npos, оголошена у класі string, означає «видалити до кінця рядка». Щоб звернутися до константи npos поза класом string потрібно писати string::npos. Таким чином, str1.erase(2); — це те ж, що str1.erase(2,string::npos);, але запис коротший і елегантніший.

Ще один приклад використання string::npos: метод find шукає підрядок у рядку й повертає позицію першого входження першого символу підрядка, а якщо підрядок не знайдено — повертає string::npos.

Приклад:

string str;
getline(cin, str);
size_t pos = str.find("123"); //або int pos = ...
if (pos == string::npos)
{
    cout <<"Підрядок 123 не знайдено!" <<endl;
}
else
{
    cout <<"Підрядок 123 починається з позиції " <<pos <<"!" <<endl;
}

Зверніть увагу: результат методу find я зберіг у змінній типу size_t. Це тому, що цей метод повертає значення цього типу. Якщо ви забули, який тип повертає метод find, або не хочете явно вказувати тип, починаючи з C++11 можна написати просто:

auto pos = str.find("123");

Зверніть увагу, що перші три методи не змінюють рядок (це показує модифікатор const наприкінці підпису методу) — це методи-запити, а два останні змінюють рядок — це методи-команди. Також зазначимо, які параметри використовуються в цих та багатьох інших функціях:

  1. Параметр str задає підрядок для пошуку або вставки. У таблиці наведені варіанти, в яких str має тип string, але підтримуються також рядки типу char*, а також просто символи типу char.
  2. Параметр pos задає позицію в рядку — місце вставки, початок видаляємої чи виділюваної області або початковий символ для пошуку. Тобто завжди вказується саме початок підрядка, а не його кінець. У більшості випадків має значення за замовчуванням, яке дорівнює нулю, тобто якщо цей параметр не вказати, пошук, видалення або виділення виконуватиметься з початку рядка.
  3. Параметр len задає довжину виділюваного або видаляємого підрядка. Зауважте, вказується саме довжина, а не позиція кінця підрядка. За замовчуванням, параметр len має значення string::npos, тобто всю решту рядка, починаючи з позиції pos і до кінця рядка. Взагалі, якщо ми вкажемо len більше, ніж є символів у рядку, починаючи з pos, то логіка буде такою: це означає всі символи до кінця рядка.
