C++ рядки (клас string)

Оголошення C++ рядків. Конструктори

Мова програмування C++ також пропонує ще один варіант рядків, реалізований відповідно до об’єктно-орієнтованого підходу. Стандартна бібліотека надає тип даних string, який є класом (class). Зауважимо, що з точки зору мови C++ клас — це визначений у C++ користувацький тип даних, що зберігає певну інформацію (дані) й має визначену поведінку (методи).

Щоб використовувати клас string, реалізований у стандартній бібліотеці C++, потрібно записати 

#include<string>

Зверніть увагу: string без букви h. Після цього клас буде доступний під іменем std::string, тобто ім'я string знаходиться у просторі імен std. Наступна стрічка

using namespace std;

позбавляє необхідності писати префікс std:: перед іменем string (а також перед cout, cin, endl та іншими іменами зі стандартної бібліотеки).

Далі у програмі оголосити C++-рядок можна так:

string str;

Правда, схоже на звичайне оголошення змінної мови Сі? Спочатку пишеться ім'я класу (нагадаємо, клас — це тип даних), а потім ім'я змінної цього типу. Конкретна змінна типу, який є класом, називається об’єктом (object) або екземпляром класу (instance of class). Тобто string — це ім'я класу, а str — ім'я об’єкта, тобто конкретного рядка.

Зверніть увагу: при такому оголошенні створюється новий рядок і для нього виділяється пам’ять. Це відбувається за допомогою спеціального методу — конструктора класу (constructor). Він присвоює щойно створеному об’єкту певне значення. Оскільки ми не вказали жодних додаткових даних, викликається конструктор без параметрів. Для класу string конструктор без параметрів створює порожній рядок (рядок із нуля символів).

Зазвичай у класу є й інші конструктори, які приймають один або кілька параметрів. Щоб скористатися таким конструктором, потрібно використовувати такий формат оголошення:

ім’я_класу ім’я_об’єкта(параметри);

наприклад

string str2("hello");

або 

string str3(str2);

У цьому випадку початкове значення рядка str2 буде "hello" (тобто рядок str2 буде створений на основі рядкової константи "hello"), а str3 отримає у початкове значення значення рядка str2.

Якщо параметрів конструктора декілька, вони пишуться через кому, як і у будь-якій функції. Наприклад, клас string має конструктор такого вигляду

string (int n, char c);

який створює рядок із n символів c. Його можна використати, наприклад, так:

string str4(100,'a');

Рядок str4 буде складатися зі 100 символів 'a'.

Якщо конструктор має лише один параметр, виклик можна записати у більш звичному вигляді, використовуючи знак =. Наприклад, рядки str2 і str3 можна оголосити так:

string str2="hello";

string str3=str2;

Усі ці виклики конструктора є неявними, оскільки у цьому разі конструктор класу автоматично викликається при створенні змінної цього класу, тобто об’єкта. Конструктор у C++ можна викликати й явно. За синтаксисом такий виклик схожий на функцію, ім'я якої співпадає з іменем класу, а отже, і з іменем конструктора. Тобто у цьому випадку конструктор повертає щойно створений об’єкт, який буде безіменним. Наприклад, якщо хочемо вивести у рядку 40 зірочок, це цілком можна зробити й без циклу за допомогою вже відомого нам конструктора рядка з двома параметрами:

cout << string(40, '*') << endl;

або при повному написанні імен (без using namespace std;):

std::cout << std::string(40, '*') << std::endl;
