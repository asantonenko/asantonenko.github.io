Розглянемо зараз мінімальну за довжиною програму, яка працюватиме у будь-якій версії мов C та C++. У будь-якій програмі, як мовою С, так і C++, має бути функція main. Виконання програми полягає у виконанні функції main. Тобто виконання програми починається з виконання першого рядка функції main — це так звана точка входу програми (англ. entry point) і закінчується поверненням з функції main. При цьому функція main в кінці свого виконання повертає значення — ціле число, що означає, чи успішно було завершено програму. За угодою, прийнятою в сучасних операційних системах, значення 0 сигналізує про успішне завершення програми, а будь-яке інше значення — що в процесі роботи програми сталася помилка.

Отже мінімальна програма складається тільки з функції main(), яка повертає значення 0 і виглядає так:

int main()
{
    return 0;
}

Перший рядок містить заголовок функції. У ньому int позначає тип даних значення, що повертається — ціле число, у круглих дужках вказані параметри функції main  — як бачите, в даному випадку параметрів немає, але порожні круглі дужки все одно потрібні. У фігурних дужках пишеться тіло функції main — тобто набір дій — операторів, які необхідно виконати. У цьому випадку оператор тільки один — оператор повернення return. Після цього ключового слова вказано саме значення — ціле число 0, яке ми хочемо повернути.

Зауважимо, що в сучасних версіях мов C (починаючи з C99) і C++ (починаючи з C++11) у функції main можна і не писати в явному вигляді оператор return — якщо його не вказати, буде повернуто значення 0. У таких версіях програму можна скоротити до:

int main()
{
}

чи навіть

int main(){}

В інших (старіших) версіях C можна було писати просто main() або void main(), але в даному курсі ми не будемо цим ніколи користуватися, щоб наші програми мовою Сі успішно запускалися і з допомогою компілятора мови C++. 

Зауважимо, що, функція main не повинна повертати нуль. Якщо ми вважаємо, що програма закінчила своє виконання невдало, або просто хочемо передати операційній системі деяке інше число, ми вільні повернути будь-яке ціле число, причому його не обов'язково задавати константою, а можна задати і складнішим виразом, наприклад, програма

int main()
{
    return 2+2;
}

 поверне операційній системі число 4.
