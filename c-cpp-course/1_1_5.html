Історія та загальна інформація

Загальну інформацію про мови C/C++ (історії їх створення, версії та можливості) можете дізнатися, наприклад, з Вікіпедії (з якої я брав частину інформації для цього вступу):

  1. https://uk.wikipedia.org/wiki/C_(мова програмування)
  2. https://uk.wikipedia.org/wiki/C++

Для тих, кого цікавить історія розвитку мов Сі та C++ докладніше, ніж описано в цьому розділі, можу рекомендувати такі онлайн джерела українською та російською мовами:

  1. Грицюк Ю. І., Рак Т. Є. Об’єктно-орієнтоване програмування мовою С++: навчальний. посібник. Львів: Вид-во ЛДУ БЖД, 2011. 404 с. — у першому розділі наведена історія виникнення мови програмування C++, включаючи причини створення мови програмування С, передумови виникнення, етапи появи та вдосконалення мови програмування C++, а також зв'язок мови програмування C++ з мовами Java і С#
    (онлайн можна знайти, наприклад, тут: http://ктеп.kiev.ua/wp-content/uploads/2019/12/Griczyuk-C.pdf).
  2. https://acm.bsu.by/wiki/C2018/История_языка_программирования_C — історія мови та витоки C, різні стандарти мови Сі, застосування Сі на практиці, порівняння з мовою C++.

Офіційна документація

Обов'язковим елементом вивчення мов C/C++ є використання офіційної документації з мов (ну чи не зовсім офіційної — але точно надійної). Для мови С++ двома такими джерелами є:

  1. http://www.cplusplus.com/ — мені він подобається більше, хоча весь ресурс викладено тільки англійською мовою — інформація більш коротка, але, як мені здається, краще структурована, гарні приклади;
  2. https://en.cppreference.com/w/ — тут інформації більше, і, як бачите, це джерело навіть частково перекладено російською мовою (але, на жаль, не українською мовою).

Справа в тому, що в книгах або, тим більше, у методичних матеріалах на кшталт цього курсу, потенційно можуть бути помилки або неточності. Більш того, далеко не всі книги та курси націлені на повний виклад всього матеріалу мовами С/C++. Така властивість повного охоплення матеріалу характерна для книг, які називаються довідниками. Але в моєму понятті використання паперових довідників вже застаріло, тоді як використання онлайн-документації дозволяє більш ефективно проводити пошук інформації.

В якості альтернативного довідника можна використовувати MSDN від Microsoft:

  1. Довідник з мови C++ | Microsoft Docs  https://docs.microsoft. com/en-us/cpp/cpp/cpp-language-reference. 

Книги

На жаль, слід зазначити, що перекладів актуальної англомовної літератури C/C++ українською мовою майже не існує, тоді як переклади російською мовою представлені досить широко. Книги ж українських чи російських авторів, на жаль, здебільшого поступаються книгам англомовних авторів. Тому нижче будуть представлені переважно переклади англомовних книг на російську.

Для новачка, на мою думку, підійдуть наступні книги:

  1. Стивен Прата. Язык программирования C. Лекции и упражнения [пер. с англ.]. - 6-е изд. - М. ; СПб. ; Киев : Вильямс, 2015.
  2. Стивен Прата. Язык программирования C++ : лекции и упражнения [пер. с англ.]. - 6-е изд. - М. ; СПб. ; К. : Вильямс, 2012.
  3. Роберт Лафоре. Объектно-ориентированное программирование в С++ - 4-е изд. - Питер, 2002. 

На жаль, ці книги не ідеальні, у Стівена Прата дуже своєрідна структура викладу матеріалу — в якій він дуже докладно розглядає кожну тему у всіх подробицях і деталях, що з одного боку добре (Ви дізнаєтеся про всі деталі), а з іншого боку якісь базові конструкції, як, наприклад, оператор if можуть пояснюватися чи не в середині книги. У цих книгах описуються нові стандарти C11 і C++11. Тоді як книга Лафоре дуже зрозуміла та логічна — але, на жаль, дещо застаріла, оскільки описує старий стандарт C++98.

Тим не менш, я рекомендую ці книги, тому що по-перше, хоча б поверхово з ними знайомий, на відміну від багатьох інших книг, по-друге вони цілком коректні і приклади непогані, і по-третє — найголовніше, вони не зарозумні і цілком зрозумілі новачкам.

Також для новачків часто рекомендують книги, з якими я, по суті, не знайомий, але які мають багато хороших відгуків:

  1. Стенли Б. Липпман, Жози Лажойе, Барбара Э. Му. Язык программирования C++. Базовый курс, 5-е издание, 1120 стр., ISBN 978-5-8459-1839-0, Вильямс, 2014.
  2. Бьярне Страуструп, Программирование: принципы и практика использования C++, 2-е издание, 1328 стр., ISBN 978-5-8459-1949-6, Вильямс, 2016.

Дані книги описують більш-менш актуальні версії мови (книга Ліппмана — C++11, Страуструпа — C++14).

Справжньою класикою є книги творців даних мов, які описують ці мови:

  1. книга Кернігана та Рітчі "Мова програмування C": перше видання: Брайан Керниган, Деннис Ритчи, Алан Фьюэр. Язык программирования Си. Задачи по языку Си. — Москва: Финансы и статистика, 1985. — 279 с. (англ:  Brian W. Kernighan, Dennis M. Ritchie The C Programming Language, Prentice-Hall,  1978), найбільш актуальне російськомовне на даний момент — Брайан Керниган, Деннис Ритчи. Язык программирования C. — Москва: Вильямс, 2015. — 304 с. — ISBN 978-5-8459-1975-5. Див. також https://uk.wikipedia.org/wiki/Мова_програмування_C_(книга) та Мова програмування C — html-версія українського перекладу другого видання книжки.
  2. книги творця мови C++ Б'ярна Страуструпа - насамперед "Мова програмування C++": перше видання: B. Stroustrup: The C++ Programming Language. Addison Wesley, ISBN 0-201-12078-X. October, 1985, тоді як найактуальніше видання Bjarne Stroustrup The C++ Programming Language (4th Edition). Addison-Wesley ISBN 978-0321563842. May 2013 — досі не перекладено російською чи українською. Див. також https://uk.wikipedia.org/wiki/Мова_програмування_С++_(книга)

Але звертаю увагу, що для непідготовленого читача дані книги можуть виявитися надто складними, особливо цим відрізнялася книга "Мова програмування C++" Б'ярна Страуструпа — новачкам не раджу з неї починати.
