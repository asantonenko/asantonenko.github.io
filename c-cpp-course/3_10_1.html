У мовах програмування вираз будується з деяких базових блоків - операндів, за допомогою деяких конструкцій - операцій та викликів функцій.

Таким чином, такі базові (атомарні) конструкції є виразами:

  1. змінні (або точніше - імена змінних);
  2. константи - як іменовані, так і літерали;
  3. у деяких окремих випадках, імена типів та функцій.

Прикладами складніших виразів є:

  1. застосування операцій (унарних, бінарних або тернарних) до більш простих виразів;
  2. деякий вираз, записаний у дужках;
  3. результати викликів функцій та застосування методів. 

 Приклади:

Нехай оголошено такі змінні:

int x, y;
double z;
char c;
bool flag;
string str;

Тоді найпростішими виразами будуть (записані по одному в рядок): 

x
y
z
c
str
10
1.5
"Вітаю!"

 Перші п'ять виразів - це імена змінних, а останні три - літерали (константи).

З найпростіших виразів можна отримувати складніші вирази за допомогою операцій, наприклад:

x++
--y
-x
2 + 2
1.5 * 3
x + 5
str + "!"
z = 1.5
cout << "Вітаю!"
cin >> y
flag ? x : 0

У перших трьох прикладах використовуються по одній унарній операції, у семи наступних - по одній бінарній і в останньому прикладі - тернарна операція. Зауважимо, що як операнди використовуються як змінні простих типів (x, y, z, flag), так та об'єкти (str, cin, cout) - змінні складніших типів, а також літерали (2, 1.5, 3, 5, "!", "Привіт!", 0).

Зауважимо, що кожна операція повертає деяке значення, тому кожен із виразів вище має деяке значення. Напишемо значення виразів для всіх виразів:

Значення виразів
Вираз Значення
x++ вихідне (незбільшене) значення змінної x
--y змінене (зменшене на 1) значення змінної y
-x число, протилежне значенню змінної x
2 + 2 4
1.5 * 3 4.5
x + 5 значення, на 5 більше, ніж значення змінної x
str + "!" значення рядка str з доданим знаком "!" наприкінці
z = 1.5 нове значення змінної z, що дорівнює 1.5
cout << "Вітаю!" потік виведення cout
cin >> y потік введення cin
flag ? x : 0 якщо flag істинна, то x, інакше 0

Деякі з цих виразів мають побічну дію - перерахуємо дані побічні ефекти:

Побічні ефекти виразів
Вираз Побічний ефект
x++ збільшення значення змінної x на 1
--y зменшення значення змінної y на 1
z = 1.5 присвоєння змінної z значення 1.5
cout << "Привет!" Виведення рядка "Привіт!" до стандартного потоку виведення
cin >> y Введення значення змінної y зі стандартного потоку введення

Перелічені п'ять виразів, які мають побічні ефекти, можна перетворити на оператори, поставивши крапку з комою в кінці виразу. Ми отримаємо наступні оператори-вирази:

x++;
--y;
z = 1.5;
cout << "Вітаю!";
cin >> y;

Зробивши це, ми ігноруємо значення даних виразів, залишаючи лише їхні побічні ефекти (зазначені в таблиці вище). Звичайно ж, й інші вирази (що не мають побічних ефектів) теж можна перетворити на оператори-висловлювання, але це не має особливого сенсу, оскільки ніяких дій не здійснюватиметься.
