В языках программирование выражение строится из некоторых базовых блоков - операндов, при помощи некоторых конструкций - операций и вызовов функций. 

Таким образом, следующие базовые (атомарные) конструкции являются выражениями:

  1. переменные (или точнее - имена переменных);
  2. константы - как именованные, так и литералы;
  3. в некоторых редких случаях, имена типов и функций.

Примерами более сложных выражений является:

  1. применение операций (унарных, бинарных или тернарной) к более простым выражениям;
  2. некоторое выражение, записанное в скобках;
  3. результаты вызовов функций, а также применения методов. 

 Примеры:

Пусть объявлены следующие переменные:

int x, y;
double z;
char c;
bool flag;
string str;

Тогда простейшими выражениями будут (записаны по одному в строку): 

x
y
z
c
str
10
1.5
"Привет!"

 Первые пять выражений - это имена переменных, а последние три - литералы (константы).

Из данных и других простейших выражений можно получать более сложные выражения при помощи операций, например:

x++
--y
-x
2 + 2
1.5 * 3
x + 5
str + "!"
z = 1.5
cout << "Привет!"
cin >> y
flag ? x : 0

В первых трех примерах используются по одной унарной операции, в семи следующих - по одной бинарной и в последнем примере - тернарная операция. Заметим, что в качестве операндов используются как переменные простых типов (x, y, z, flag), так и объекты (str, cin, cout) - переменные более сложных типов, а также литералы (2, 1.5, 3, 5, "!", "Привет!", 0).

Заметим, что каждая операция возвращает некоторое значение, поэтому каждое из выражений выше имеет некоторое значение. Напишем значения выражений для всех выражений:

Значение выражений
Выражение Значение
x++ исходное (неувеличенное) значение переменной x
--y измененное (уменьшенное на 1) значение переменной y
-x число, противоположное значению переменной x
2 + 2 4
1.5 * 3 4.5
x + 5 значение, на 5 большее, чем значение переменной x
str + "!" значение строки str с добавленным знаком "!" в конце
z = 1.5 значение переменной z, равное 1.5
cout << "Привет!" поток вывода cout
cin >> y поток ввода cin
flag ? x : 0 если flag истинна, то x, иначе 0

Некоторые из данных выражений имеют побочное действие - перечислим данные побочные эффекты:

Побочные эффекты выражений
Выражение Побочный эффект
x++ увеличение значения переменной x на 1
--y уменьшение значения переменной y на 1
z = 1.5 присвоение переменной z значения 1.5
cout << "Привет!" Вывод строки "Привет!" в стандартный поток вывода
cin >> y Ввод значения переменной y со стандартного потока ввода

Перечисленные пять выражений, которые имеют побочные эффекты можно превратить в операторы, поставив точку с запятой в конце выражения. Мы получим следующие операторы-выражения:

x++;
--y;
z = 1.5;
cout << "Привет!";
cin >> y;

Сделав это, мы игнорируем значения данных выражений, оставляя только их побочные эффекты (указанные в таблице выше). Конечно же и другие выражения (не имеющие побочных эффектов) тоже можно превратить в операторы-выражения, но это не имеет особого смысла, так как никаких действий осуществляться не будет.
