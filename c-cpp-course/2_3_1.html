У мовах C і C++ значення деяких слів жорстко фіксовано, вони застосовуються для опису фундаментальних типів (таких, як int і double), опис операторів мови (наприклад, if або return) та інших потреб мови. Такі слова називаються ключовими словами мови програмування. Не можна використовувати ключові слова як імена змінних, функції та інші ідентифікатори.

Набір ключових слів різний залежно від мови та її версії. У мові C менше ключових слів, ніж у мові C++, причому у новіших версіях мов з'являються нові і нові ключові слова. Це обмежує сумісність мов. Наприклад, у мові C цілком можна описати змінну з ім'ям class, але в мові C++ слово class є ключовим. У тексті цього курсу ключові слова набрані жирним шрифтом, у тексті програм у популярних середовищах ключові слова виділяються також жирним шрифтом або кольором. Придивившись дуже уважно до вже розглянутого прикладу програми "Привіт, світе!", Ви можете побачити, що ключові слова int і return виділені іншим кольором, ніж main та printf (нажаль кольори дуже схожі, але, все ж таки, різні).

/* Програма Привіт, світе! */
#include <stdio.h>

int main()
{
    printf("Привіт, світе!\n");
    return 0;
}

У початковому стандарті мови C - C89 присутні наступні ключові слова (взято з джерела wikipedia C (мова програмування)):

Ключові слова Призначення
sizeof Отримання розміру об'єкта чи типа на етапі компіляції
typedef Завдання альтернативного імені типу
auto, register Підказки компілятору щодо місця зберігання змінних
extern Вказівка компілятору шукати об'єкт поза поточним файлом
static Оголошення статичного об'єкта
void Маркер відсутності значення; у покажчиках означає довільні дані
char, short, int, long Цілочисленні типи та модифікатори їхнього розміру
signed, unsigned     Модифікатори цілих типів, що визначають їх як знакові чи беззнакові
float, double Дійсні типи даних
const Модифікатор, що унеможливлює зміну деякого значення
volatile Вказівка ​​компілятору на можливість зміни значення змінної ззовні
struct Тип даних — структура з набором полів
enum Тип даних, що зберігає одне з набору цілих значень
union Тип даних, у якому можна зберігати дані одного з декількох обраних типів
do, for, while Оператори циклу
if, else Умовний оператор
switch, case, default Оператор вибору за цілим параметром
break, continue Оператори переривання циклу
goto Оператор безумовного переходу
return Повернення з функції
