Компілятори

Мови C і C++ є компільованими мовами. Це означає, що для того, щоб запустити програму мовою C або C++, необхідна спеціальна програма — компілятор мови C або C++, відповідно, завданням якої є переклад вихідного коду мовою C/C++, написаного зазвичай людиною (програмістом) у машинний код для певної архітектури. Безпосередньо виконуватися на комп'ютері може лише програма в машинних кодах архітектури, що відповідає використовуваному в комп'ютері процесору. Прикладами таких архітектур є x86, що використовується в сучасних комп'ютерах (або, точніше, її 32-бітний варіант x86, а також її 64-бітовий варіант x86-64) і ARM, що використовується в основному смартфонах і планшетах, а також у Smart TV та деяких ноутбуках від Apple. Існують і інші архітектури, наприклад AVR, що використовується в мікроконтролерах, що керують різними побутовими приладами та технологічними процесами на виробництві.

Виділимо основні версії компіляторів С і С++: популярними комерційними компіляторами під Windows є Visual C і Visual C++, що входять до складу середовища Microsoft Visual Studio, тоді як найбільш популярним компілятором з відкритим вихідним кодом на даний момент є gcc (GNU Compiler Collection) та його варіант для мови C++ — який називається g++. Також останнім часом набирає популярності компілятор з відкритим вихідним кодом clang.

На платформі stepik для компіляції C та C++ використовуються компілятори сімейства gcc, тому докладніше розглянемо їх. GNU Compiler Collection (зазвичай використовується скорочення GCC) — набір компіляторів для різних мов програмування, розроблений в рамках проекту GNU.

GCC є вільним програмним забезпеченням, поширюється фондом вільного програмного забезпечення (FSF) на умовах GNU GPL та GNU LGPL та є ключовим компонентом GNU toolchain. Він використовується як стандартний компілятор для вільних UNIX-подібних операційних систем.

Спочатку GCC розшифровувався як GNU C Compiler та підтримував лише мову Сі. Пізніше GCC був розширений для компіляції вихідних кодів мовами програмування, як C++, Objective-C, Java, Фортран, Ada, Go, GAS і D.

На платформі stepik на кінець 2020 року можна було вибрати три компілятори сімейства C/C++:

  1. C — використовується gcc 6.3.0, стандарт C99;
  2. C++ — використовується g++ 6.3.0, стандарт у параметрах компілятора не вказаний, мабуть C++14;
  3. C++11 — використовується g++ 6.3.0, стандарт C++11.

Нещодавно на платформу stepik був доданий компілятор C++20 (використовується  g++ 10.2.1).
