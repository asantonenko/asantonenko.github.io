У мовах C і C++ є такі бінарні операції порівняння

Знак Операція Приклад виразу Результат
< менше 1 < 2 true
<= менше або дорівнює (не більше) 1 <= 2 true
> більше 3 > 3 false
>= більше або дорівнює (не менше) 3 >= 3 true
== дорівнює 2 * 2 == 4 true
!= не дорівнює 3 - 2 != 1 false

У мові С++ операції порівняння повертають логічне значення true, якщо числа знаходяться в цьому відношенні і false, інакше. У мові C замість цього повертаються значення 1 (позначає істину) або 0 (позначає хибу).  Нагадаємо, що у старому стандарті мови C немає типу bool.
