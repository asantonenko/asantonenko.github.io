Введення і виведення рядків

Для виведення рядків використовується звичайна конструкція cout << str;, де str — це змінна типу char * або string. Для введення можна використовувати cin >> str, але оскільки роздільниками в цьому випадку є і пробіли, і переходи на новий рядок, можуть виникнути неприємні сюрпризи:

string FIO;

cout <<"Введіть Прізвище Ім'я та По батькові:" <<endl;

cin >>FIO; //Вводимо: Іванов Іван Іванович

cout <<"Вітаю, " <<FIO <<"!" <<endl;

// Отримали вивід: Вітаю, Іванов!

// Ім'я та по батькові загубилися

Щоб прочитати рядок повністю, до символу переходу на новий рядок (Enter), необхідно скористатися функцією getline у такому вигляді:

string FIO;

cout <<"Введіть Прізвище Ім'я та По батькові:" <<endl;

getline(cin, FIO); //Вводимо: Іванов Іван Іванович

cout <<"Вітаю, " <<FIO <<"!" <<endl;

// Отримуємо: Вітаю, Іванов Іван Іванович!

Таким чином, функція getline — це аналог функції gets. Але є одна суттєва відмінність. Якщо у функції gets ми введемо з клавіатури більше символів, ніж виділено пам'яті, результат буде вкрай негативним. Для класу string пам'ять виділяється динамічно, під час виконання програми, повністю прозоро (непомітно) для програміста. Втім, для рядкової змінної str можна дізнатись, скільки виділено пам'яті для неї за допомогою методу str.capacity() і навіть зарезервувати місце заздалегідь за допомогою str.reserve(кількість). При цьому такого неприємного явища як пошкодження пам'яті виникнути не може, на відміну від функції gets.

Зверніть увагу, що використання функції getline пов'язане з однією неприємною особливістю. Запустімо наступний код:

int age;

cout <<"Введіть Ваш вік:" <<endl;

cin >>age; //Вводимо 25 і натискаємо Enter

string FIO;

cout <<"Введіть Прізвище Ім'я та По батькові:" <<endl;

getline(cin, FIO); //Навіть не дає нічого ввести!

cout <<"Вітаю, " <<FIO <<"!" <<endl;

// Отримуємо: Вітаю, !

У змінну FIO потрапляє порожній рядок! Чому? Справа в тому, що умовний курсор вводу знаходиться у рядку, де ми ввели значення віку (25) — одразу після числа 25. Отже, від цього місця до кінця рядка (до символу переходу на новий рядок) знаходиться порожній рядок, який потрапляє у змінну FIO, на чому робота getline і завершується. Це не помилка — але не те, що нам потрібно. Тому перед виконанням getline використаємо конструкцію cin >>ws;. Вона пропустить усі пробільні символи, у тому числі й переходи на новий рядок, і введення розпочнеться з нового рядка (з побічним ефектом: введений рядок із ПІБ ніколи не починатиметься з пробілу).

int age;

cout <<"Введіть Ваш вік:" <<endl;

cin >>age; //Вводимо 25 і натискаємо Enter

string FIO;

cout <<"Введіть Прізвище Ім'я та По батькові:" <<endl;

cin >>ws; //вирішили проблему з порожнім рядком при введенні

getline(cin, FIO); //Вводимо: Іванов Іван Іванович

cout <<"Вітаю, " <<FIO <<"!" <<endl;

// Отримуємо: Вітаю, Іванов Іван Іванович!
