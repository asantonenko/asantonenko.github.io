Операції над C++ рядками

Звісно, створені вами змінні типу рядок (string) можна згодом змінювати (якщо ви не створили рядкову константу типу const string). Оператор = означає присвоєння рядків, оператор + — конкатенацію (об’єднання) рядків. Як і для Сі-рядків, для отримання i-го символу рядка str можна записати str[i]. Наприклад, після такого фрагменту:

string str;

string str2 = "hello";

string str3 = str2;

string str4(100,'a');

str = "world!";

str3 = str2 + ' ' + str;

str4[0] = 'b';

рядок str буде дорівнювати "world!", рядок str2 залишиться рівним "hello", рядок str3 набуде значення "hello world!", а рядок str4 складатиметься з букви 'b' і далі 99 букв 'a'.

Слід пам’ятати, що оператор = з правого боку може містити не лише значення типу string, а й Сі-рядок типу char*, а одним з операндів оператора + може бути тип char або char*. Але хоча б один з операндів + повинен мати тип string, тобто якщо str1 та str2 мають тип string, то

а) допустимі конструкції:

string plus1 = str1 + str2;

string plus2 = str1 + "!!!";

string plus3 = str1 + '!';

string plus4 = "!!!" + str2;

string plus5 = '!' + str2;

б) недопустимі конструкції:

string wrong1 = "!!!" + "!!!"; //помилка компіляції

string wrong2 = '!' + "!!!";

string wrong3 = "!!!" + '!';

string wrong4 = '!' + '!';

При цьому друга та третя конструкції скомпілюються успішно, але працюватимуть не як конкатенація, як ми, можливо, очікували, а беззмістовно. Тут видно, що виводяться дуже дивні значення (якщо згадаєте про адресну арифметику, то зрозумієте чому).

Щоб виправити помилки (компіляції або смислові), зазначені вище, можна перетворити символ типу char або рядок типу char* на об’єкт класу string, наприклад так:

string fixed1 = string("!!!") + "!!!"; //помилки виправлені

string fixed2 = string(1,'!') + "!!!";

string fixed3 = string("!!!") + '!';

string fixed4 = string(1,'!') + '!';

Тут використано конструктор, який створює рядок типу string із значення char*, і конструктор із двома параметрами, який створює рядок типу string з одного символу типу char.

Як і для числових даних, запис str1 += str2; еквівалентний за змістом запису str1 = str1 + str2;, тобто додає рядок str2 у кінець рядка str1. Але зверніть увагу, що цілком можливо, що оператор str1 += str2; спрацює швидше, оскільки йому не потрібно створювати новий рядок-конкатенацію двох рядків і потім записувати її на місце рядка str1.
