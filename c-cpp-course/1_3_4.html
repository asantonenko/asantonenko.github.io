Що якщо Ви хочете перевірити, як працює деяка програма деякою мовою програмування? Тоді Вам знадобиться середовище для редагування, компіляції та запуску програми - так зване інтегроване середовище розробки (integrated development environment , IDE). Подібні середовища є як офлайнові - у вигляді програми для Windows/Linux/Mac OS чи Android/iOS - наприклад, кросплатформова безкоштовна Code::Blocks або також кросплатформова і безкоштовна Microsoft Visual Studio Code. Але так як Ви зараз, очевидно використовуєте браузер (ну або мобільний додаток stepik), то логічніше познайомитися з онлайн IDE, для роботи яких потрібний тільки браузер. Мабуть, однією з найзручніших і найфункціональніших онлайн IDE є OnlineGDB. Давайте візьмемо нашу програму

// Програма Привіт, світе!
#include <iostream>

int main()
{
    std::cout << "Привіт, світе!" << std::endl;
    return 0;
}

і перевіримо її роботу на OnlineGDB, cpp.sh , ideone  або Wandbox. Тут наведені прямі посилання на цю програму на цих платформах. Виконайте програму "Привіт, світе!" послідовно за даними чотирьом посиланням (для знайомства з даними системами). Запустити програму на OnlineGDB дуже просто - натисніть зелену кнопку з написом Run. На ресурсі cpp.sh потрібно натиснути кнопку Run, а на ideone результат відразу ж буде у підписаному вікні "stdout". Якщо ж Ви хочете внести зміни до цієї програми, то натискаєте кнопку fork (в ideone) / Fork this (в OnlineGDB), створюючи нову версію, а потім зелену кнопку Run. Після запуску програми на ideone результат дивитися в віконці "stdout" - і оскільки тепер це вже Ваша програма, можна не створювати нову версію (fork) знову, а натиснути кнопку редагувати і потім кнопку Submit для запуску програми (робота з ideone трохи заплутана, на жаль)! В onlineGDB після того як Ви зробили fork деякої програми, її теж можна редагувати, зберігати або розповсюджувати далі за допомогою кнопки Share. На Wandbox Ви відразу можете побачити результати роботи програми (це те, що в чорному вікні написано білим кольором, саме білим!, а не іншим), а також змінити програму та натиснути кнопку Run для запуску зміненої програми.

Майже ту ж програму, але вже написану з використанням конструкції using namespace std;:

// Програма Привіт, світе!
#include <iostream>

using namespace std;

int main()
{
    cout << "Привіт, світе!" << endl;
    return 0;
}

можна знайти за посиланнями на OnlineGDB, cpp.sh или ideone чи Wandbox.

Підказка: Якщо Ви налаштовані серйозно використовувати OnlineGDB або ideone, дуже раджу зареєструватися на цих системах. Втім, в систему OnlineGDB можна заходити також через Google, Facebook або Github аккаунт. Реєстрація дозволить Вам зберігати програми у своєму обліковому записі та переглядати та сортувати список написаних Вами програм — що є дуже зручним — у будь-якому місці світу, де є Інтернет, у Вас є доступ до всіх написаних Вами програм. Також підкреслю, що і в системі stepik, у будь-якому місці світу, де є Інтернет, у Вас є доступ до всіх Ваших рішень (як успішних, так і помилкових) завдань даної платформи.
