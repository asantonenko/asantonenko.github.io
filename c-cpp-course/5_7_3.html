Опишем базовый функционал контейнера map, взяв за основу статью https://ru.wikipedia.org/wiki/Map_(C++).

Стандартные методы класса map дают возможность пользователю осуществлять операции поиска, вставки и обхода контейнера унифицированным образом. 

Название метода или операции Функции Пример использования метода или операции
size() Возвращает количество элементов в контейнере. cout <<"Количество слов в словаре"
 <<dict.size() <<endl;
empty() Возвращает true если контейнер пуст, иначе false.

if (dict.empty()) {
  cout <<"Словарь пуст!"
       <<endl;
}
find(k) Возвращает итератор, указывающий на значение, соответствующее значению ключа k.
Если такого ключа в контейнере нет, то возвращается итератор end().
Если ключ найден, то возвращается итератор на пару ключ - значение.
Пусть мы присвоим итератор переменной it, тогда
it->first будет обозначать ключ (равный k),
 а it->second - значение.

string word;
cin >> word;
auto it = dict.find(word);
if (it == dict.end()) {
   cout <<word
        <<" не найдено!"
        <<endl;
} else {
   cout <<word 
        <<" переводится "
        <<it->second
        <<endl;
}
operator[k] Возвращает ссылку на значение, соответствующее ключу k.
Если такого ключа не существует, то он создаётся.		 dict["she"] = "она";
insert(pair(k,v)) Вставляет в контейнер пару {k,v}, возвращая адрес его позиции. 
Если ключ k уже присутствовал в контейнере map, то метод игнорируется.		 dict.insert({"hi", "привет"});
erase(k) Удаляет из контейнера элемент с ключом k. dict.erase("zoo");
erase(p) Удаляет из контейнера элемент, на который указывает итератор p. dict.erase(dict.begin());
begin() Возвращает итератор на начало контейнера. auto it = dict.begin();
end() Возвращает итератор на конец контейнера. см. пример на find(k)
